Pibrary

Pibrary is a project to put a library management system onto a Raspberry Pi. It is aimed at schools and is intended as a near-zero cost investment.

Pibrary consists of two components:

  • Orac: a RESTful webservice that manages the library catalogue
  • Zen: a web application providing a full Angular.js and Bootstrap UI and member management service

Both components are built on Node.js using the Express middleware module and use MongoDB for database services. Zen utilises the Jade server side templating engine.

Fixtures & Scores

Fixtures & Scores is an Angular.js directive to implement a display of upcoming fixtures and recent results, for a sports team.

The directive retrieves JSON formatted data, and displays two tables; one for upcoming fixtures, one for results. An Excel spreadsheet provides a front end, allowing users to enter their upcoming match fixtures and recent scores. When saved, the Excel sheet saves a copy of the data in JSON format. This can be saved into Dropbox, and retrieved by the directive.

<fixtures src="https://your_url_here/fixtures.json?callback=JSON_CALLBACK" />

School Book Store

A prototype for a school to implement a web based book store, allowing parents to buy their childrens' books for the year. The books are displayed by class, along with guidance as to what books can be reused, what may be available second hand and allow the buyer to specify whether or not they would like the books covered.

The server side is Node.js based using Express middleware and Jade templates, with an Angular.js client front end.
